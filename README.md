# 加入微软俱乐部开发组

## 准备
如果想要加入微软俱乐部开发组，你需要满足以下条件：  
1. 你打算成为，或者已经是微软俱乐部部员
2. 对至少一门语言比较熟练
3. 承诺遵守我们的代码风格
4. 拥有`GitHub`账号，并且能较快地访问`GitHub`和`Google`

## 步骤
1. 去我们的测试页面，并按照指示回答问题
2. 等待开发组给你发送的邮件
3. 成为开发组成员！
